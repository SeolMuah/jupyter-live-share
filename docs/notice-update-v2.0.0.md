# [공지] Jupyter Live Share 업데이트 안내 (v2.0.0)

안녕하세요.

Jupyter Live Share가 **v2.0.0으로 업데이트**되었습니다.
**텍스트 파일 공유**, **커서/선택 공유 강화**, **다크모드** 등 여러 기능이 추가되었습니다.

**1~2분이면 끝나니** 수업 전에 꼭 업데이트해 주세요.

---

## 업데이트 내용

### 새로운 기능

- **텍스트 파일 공유**: `.py`, `.txt`, `.md` 등 텍스트 파일도 실시간 공유 가능 (구문 하이라이팅, Markdown 렌더링 지원)
- **텍스트 파일 커서/선택 공유**: 노트북뿐 아니라 텍스트 파일에서도 선생님 커서 위치, 라인 하이라이트, 선택 영역 실시간 표시
- **선생님 이름 설정**: 세션 시작 전 사이드바에서 표시 이름 변경 가능 (기본: "Teacher")
- **다크/라이트 모드**: 뷰어에서 테마 전환 지원

### 개선 사항

- **파일 전환 안정성**: 노트북 ↔ 텍스트 파일 전환 시 안정성 향상
- **채팅 선생님 메시지 강조**: 선생님 메시지가 초록색 배경으로 구분 표시 (브라우저 호환성 개선)
- **뱃지 리셋 수정**: 사이드바/Viewer Chat 뱃지가 정상적으로 초기화
- **설문 투표 강화**: 1인 1표, 재투표 불가

---

## 업데이트 방법

이전에 설치해 본 적 있으면 **다운로드 → 설치만 하면 됩니다.**

### 1. 새 버전 다운로드
- 아래 링크에서 `.vsix` 파일 다운로드:
  https://github.com/SeolMuah/jupyter-live-share/releases/latest

### 2. VS Code에서 설치
- VS Code → 좌측 Extensions (`Ctrl+Shift+X`) → 상단 **···** → **"Install from VSIX..."** → 다운받은 파일 선택

### 3. VS Code 재시작
- 설치 후 **VS Code를 한 번 재시작**해 주세요

끝입니다!
