# [공지] Jupyter Live Share 업데이트 안내 (v0.2.0)

안녕하세요.

Jupyter Live Share가 **v0.2.0으로 업데이트**되었습니다.
**선생님 커서 공유 기능**이 추가되고, 화면 동기화 안정성이 크게 개선되었습니다.

**1~2분이면 끝나니** 수업 전에 꼭 업데이트해 주세요.

---

## 업데이트 내용

- 선생님 커서 위치가 실시간으로 표시됩니다 (파란색 커서 + 라인 하이라이트)
- 선생님이 드래그한 영역이 노란색으로 표시됩니다
- 화면 동기화가 더 빠르고 안정적으로 개선되었습니다
- 표(DataFrame) 출력이 중복 표시되던 문제가 수정되었습니다

---

## 업데이트 방법

이전에 설치해 본 적 있으면 **다운로드 → 설치만 하면 됩니다.**

### 1. 새 버전 다운로드
- 아래 링크에서 `.vsix` 파일 다운로드:
  https://github.com/SeolMuah/jupyter-live-share/releases/latest

### 2. VS Code에서 설치
- VS Code → 좌측 Extensions (`Ctrl+Shift+X`) → 상단 **···** → **"Install from VSIX..."** → 다운받은 파일 선택

### 3. VS Code 재시작
- 설치 후 **VS Code를 한 번 재시작**해 주세요

끝입니다!
