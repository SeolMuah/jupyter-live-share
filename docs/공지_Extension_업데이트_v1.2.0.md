# [공지] Jupyter Live Share 업데이트 안내 (v1.2.0)

안녕하세요.

Jupyter Live Share가 **v1.2.0으로 업데이트**되었습니다.
**채팅 분리**, **Markdown 원본 전환**, **채팅 뱃지** 등 여러 개선이 있습니다.

**1~2분이면 끝나니** 수업 전에 꼭 업데이트해 주세요.

---

## 업데이트 내용

### 주요 변경

- **VS Code Viewer 채팅 분리**: Open Viewer 사용 시 채팅이 하단 "Viewer Chat" 패널로 분리
- **Markdown 원본/렌더링 전환**: `.md` 파일에서 선생님 커서 활성 시 원본 텍스트 표시, idle 시 렌더링 복귀
- **선생님 메시지 강조**: 선생님 채팅 메시지가 초록색 배경으로 구분 표시
- **파일 없이 세션 시작**: 파일을 열지 않은 상태에서도 세션 시작 가능
- **사이드바 채팅 뱃지**: 사이드바를 안 보고 있을 때 학생 메시지가 오면 아이콘에 숫자 뱃지 표시
- **안읽은 메시지 뱃지**: Viewer Chat 탭에 새 메시지 숫자 표시
- **설문 투표**: 1인 1표, 한 번 투표하면 재투표 불가 (버튼 비활성화)

---

## 업데이트 방법

이전에 설치해 본 적 있으면 **다운로드 → 설치만 하면 됩니다.**

### 1. 새 버전 다운로드
- 아래 링크에서 `.vsix` 파일 다운로드:
  https://github.com/SeolMuah/jupyter-live-share/releases/latest

### 2. VS Code에서 설치
- VS Code → 좌측 Extensions (`Ctrl+Shift+X`) → 상단 **···** → **"Install from VSIX..."** → 다운받은 파일 선택

### 3. VS Code 재시작
- 설치 후 **VS Code를 한 번 재시작**해 주세요

끝입니다!
