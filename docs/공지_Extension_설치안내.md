# [공지] 수업 실시간 공유 Extension 설치 안내

안녕하세요.

이번 수업부터 **Jupyter Live Share**를 사용합니다.
선생님이 VS Code에서 작성하는 코드와 실행 결과를 **여러분의 화면에서 실시간으로** 볼 수 있고, **채팅**과 **설문조사**로 수업 중 소통도 가능합니다.

---

## ▶ 수업 전에 꼭 해주세요

### 1. VS Code 설치 (이미 설치되어 있으면 건너뛰세요)
- https://code.visualstudio.com

### 2. Extension 설치
- 아래 링크에서 VSIX 파일 다운로드:
  https://github.com/SeolMuah/jupyter-live-share/releases/latest
- VS Code → 좌측 Extensions (`Ctrl+Shift+X`) → 상단 **···** → "Install from VSIX..." → 다운로드한 파일 선택
- **설치 후 VS Code 재시작** (필수)

### 3. 설치 확인
- `Ctrl+Shift+P` → "Jupyter Live Share" 입력 → 명령어가 나타나면 완료

---

## ▶ 수업 시간에 접속하는 법

선생님이 수업 시작 시 **URL을 공유**합니다.

- **VS Code로 보기**: `Ctrl+Shift+P` → "Open Viewer" → URL 입력
- **브라우저로 보기**: Chrome/Edge에서 URL 직접 접속 (설치 없이 가능, 모바일도 OK)

접속 후 이름을 입력하면 수업 화면이 표시됩니다.
